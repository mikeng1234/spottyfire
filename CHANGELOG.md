# Changelog

## 1.0.0 (2024-12-01)

### Features
- Initial release
- 6 chart types: BarChart, ScatterPlot, LineChart, PieChart, HeatMap, DataTable
- Cross-chart marking (click, shift+click, ctrl+click, lasso select)
- 6 built-in themes: Midnight, Arctic, Forest, Sunset, Corporate, Neon
- Custom theme support with runtime switching
- DataStore with CSV/JSON loading
- FilterEngine with value and range filters
- FormulaEngine with 30+ built-in functions and window aggregates
- FilterPanel, FormulaBar, and Toolbar UI components
- ChartWrapper with panel styling and animations
- CSV export (all data or marked only)
- 200-row Filipino employee sample dataset
- 6 example files including Streamlit embedding
